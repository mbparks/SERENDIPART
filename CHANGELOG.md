# Changelog

## 1.5
- Rebuilt Discover as a three-step draw workflow: type, quantity, and source pool.
- Added 1, 2, 4, 6, and 8-part batch draws.
- Added 1–4 simultaneous Collision Lab pairs with numbered prompts and notes.
- Added Supplier Sweep to draw one product from every eligible source.
- Added batch save, pair save, clear draw, active-source chips, and in-batch duplicate protection.
- Moved source weighting, proxy configuration, and category scopes into expandable advanced settings.
- Changed the migrated default from one collision pair to four Mixed Parts.

## 1.4
- Added Supplier Intelligence comparison workspace.
- Added local and live offer normalization, lifecycle warnings, and source reference counts.
- Added same-component comparison across distributor catalogs.
- Added direct linked part numbers throughout the instrument.

## 1.3
- Added Newark / element14 and TME.
- Added Adafruit, SparkFun, Pololu, and AutomationDirect packs.
- Added source-specific category scopes and pack controls.

## 1.2
- Added DigiKey and MISUMI.
- Added weighted source selection, new discipline pools, and cross-distributor duplicate avoidance.
- Added normalized multi-source proxy adapters.
